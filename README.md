# Bootstrap 5 CSS only

A NPM component for [Bootstrap 5](http://getbootstrap.com/) Compiled CSS and font libraries only. No javascript.

## Installation:

#### NPM (3.3.5+)

```bash
$ npm install bootstrap-5-css-only
```
